# SMS3 Public Deploy

This repository is generated from the SMS3 development repository and is only used for deployment.
The source of truth is the private/development `sms3` repository.
Maintainer deployment notes and host-specific files live in the development repo's `deploy` symlink.
Do not develop SMS3 here; manual edits may be overwritten by the deploy packaging script.

SMS3 runs locally in the browser; sequence and table inputs are not uploaded to a server.
